# Contributing to DX Terminal

Thanks for your interest in contributing\!

## Getting Started

```bash
git clone https://github.com/pdaxt/dx-terminal
cd dx-terminal
cargo build
cargo test
```

## Pull Requests

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Run `cargo test` and `cargo clippy`
5. Submit a PR

## Issues

Found a bug? Open an issue with steps to reproduce.
